## 🚀 Featured Projects

### 📚 [EduCycle – P2P Academic Book Exchange](https://github.com/trhlow/EDUCYCLE)
A **comprehensive peer-to-peer platform** for students to buy, sell, and exchange academic books and study materials with built-in chat, rating system, and secure transaction handling.

**Tech Stack:**
- Backend: Java 17, Spring Boot 3.2.5, Spring Security, Spring Data JPA, Flyway
- Frontend: React 19, Vite 7, React Router, Axios, Context API
- Database: PostgreSQL 16
- Infrastructure: Docker, Docker Compose, nginx, GitHub Actions CI/CD
- Real-time: WebSocket STOMP + SockJS

**Key Features:**
- 🔐 Multi-auth system (email `.edu.vn`, Google, Microsoft OAuth)
- 📦 Product management with image upload & approval workflow
- 💳 P2P transaction system with OTP verification
- 💬 Real-time chat within transactions
- ⭐ 5-star rating & reputation system
- 🤖 AI chatbot assistant (Claude integration)
- 📊 Admin dashboard with analytics
- 🔄 Automated CI/CD pipeline

**Repository:** [github.com/trhlow/EDUCYCLE](https://github.com/trhlow/EDUCYCLE)
